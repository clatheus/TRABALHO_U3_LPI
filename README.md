# TRABALHO_U3_LPI
Esse projeto consiste no trabalho da unidade 3 da disciplina Linguagem de Programação I. É um sistema que gerencia o tráfego entre cidades, considerando diferentes tipos de trajeto e meios de transporte. O sistema permite o cadastro de cidades, trajetos, transportes e passageiros, além de calcular e simular viagens entre as cidades.
